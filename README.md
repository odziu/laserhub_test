# Test task

## Setup

Install npm modules into 'test' folder using command in terminal:

    npm i

## Run test

To tart test run use next command:

    node --max_old_space_size=4096 node_modules\protractor\built\cli.js conf.js --specs specs/test_spec.js

or

    npm test
